## Run Locally

The app is served from the `SanjayPandeyPortfolio` folder.

```bash
cd /Users/sanjaypandey/LLM/SanjayPandeyPortfolio
python3 -m http.server 4173
```

Then open:

```text
http://127.0.0.1:4173
```
